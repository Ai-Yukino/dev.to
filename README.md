# dev.to

Source for my blog posts on [dev.to/uwu](https://dev.to/uwu).

## Format

```
##-##-##-{Day}
|___1   (post number)
|   |   xxxxxxxxx_##-##.png (Banner image: YouTube ID, timestamp)
|   |   meta.md (post title, tags)
|   |   source.md (post body)
|___2
...
|___3
...
.
.
.
```
